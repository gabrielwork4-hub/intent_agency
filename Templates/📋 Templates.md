## ⚙️ Configuração do Templater (uma vez)
1. Settings → Templater → **Template folder location** = `Templates`
2. (Opcional) **Trigger Templater on new file creation** = ON
3. (Opcional) Atalho para "Templater: Create new note from template"

## Como usar
- `Ctrl + P` → "Templater: Create new note from template" → escolher o modelo.
- Os campos `<% %>` (ex. data) são preenchidos automaticamente pelo Templater.
- Ajustar o conteúdo conforme o cliente/peça.
