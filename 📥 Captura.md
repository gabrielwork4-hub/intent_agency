# 📥 Captura (Inbox)

> Lugar para jogar ideias, keywords, insights e tarefas **no calor do momento**, sem
> interromper o fluxo. Processar depois: cada item vira nota/tarefa no lugar certo, ou é descartado.
> Com o plugin **QuickAdd**, dá pra capturar aqui com um atalho de qualquer lugar.

## ⚙️ Setup do QuickAdd (quando instalar)
1. Settings → QuickAdd → adicionar uma **Capture** choice.
2. Apontar para este arquivo (`📥 Captura.md`), inserir no fim da seção "A processar".
3. (Opcional) atalho de teclado para o comando QuickAdd.
