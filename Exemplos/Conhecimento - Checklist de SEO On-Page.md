# Conhecimento - Checklist de SEO On-Page

> Esta é uma nota de **conhecimento reutilizável**. Repare que ela é citada por

## Checklist
- [ ] Title tag única, com palavra-chave principal (até ~60 caracteres)
- [ ] Meta description atraente (até ~155 caracteres)
- [ ] Um único `<h1>` por página
- [ ] URL curta e descritiva
- [ ] Imagens com texto `alt`
- [ ] Links internos para páginas relacionadas
- [ ] Conteúdo respondendo à intenção de busca
- [ ] Página rápida (Core Web Vitals)
